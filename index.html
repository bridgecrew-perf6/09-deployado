Práctica deployado

¿Qué es un deploy en programación?

Deploy es un término famoso entre los desarrolladores web. Puede significar muchas cosas, dependiendo del ambiente y de la tecnología usada. Sin embargo, los significados que más se refieren a la práctica y pueden resumir su función son: implantar, colocar en posición, habilitar para uso o, simplemente, publicar.

Se puede pensar a git como una estanteria a la que se le pueden ir agregando cajas con cosas adentro. Estas cajas van a contener las diferentes versiones de nuestro sitio que vayamos haciendo cada vez que hagamos cambios en él.

git init

Para inicializar el repositorio.
*OBSEQUIO A MI TIA BUSCO UNA CAJITA PARA ENVOLVER EL REGALO, LO ESTOY PREPARANDO, LO ESTOY INICIALIZANDO*

Para poder subir cambios a nuestro repositorio tenemos que tener cambios para guardar. Una vez que tengamos cambios hechos.

git add .

Estamos guardando una nueva versión de nuestro sitio que incluye los nuevos cambios dentro de dicha caja.
*METER LOS REGALITOS A LA CAJA*

git commit -m "comentario con la descripcion del cambio"

Cerrar las cajas con nuestros cambios y ponerle una cinta con un texto que describa su contenido, asi podemos saber que cambios tiene si decidimos volver a ella en el futuro.
*PONERLE LA ESTAMPITA SI ES SU CUMPLEAÑOS FELIZ CUMPLEAÑOS*

*COMO SABE EL CARTERO QUE EL REPO DE MI COMPU LO TIENE QUE ENTREGAR EN UN REPOSITORIO REMOTO -> LE DOY LA DIRECCION"

git remote add origin (URL DE SU REPO)

*VAS A IR A ENTREGAR ESTE REPOSITORIO A ESA DIRECCION Y VAN A QUEDAR ENLAZADOS LA DIRECCION DE MI TIA CON ESTE REPOSITORIO CON ESTE REGALO* SIEMPRE VAN A QUEDAR UNIDOS HASTA QUE YO ELIMINE LA CARPETITA DE GIT O HASTA QUE YO BORRE ESA CARPETA*
Ya está vinculado nuestro repositorio

git push

Comienza a subir nuestros cambios a github. Seria como poner la caja ya lista para guardar en la estanteria de git. Si quisieramos guardar mas cambios tendríamos que repetir todo el proceso nuevamente Esto permite que tengamos las versiones de nuestro sitio organizadas temporalmente pudiendo volver a ella cuando necesitemos.
*YA TIENES TODO LISTO, VETE A ENTREGARLO*
